# CPP Module 04

## 📚 Description
This project introduces **subtype polymorphism, abstract classes, and interfaces in C++**.
The main goal is to understand how to design extensible systems using inheritance and virtual functions, enabling runtime polymorphism and clean separation of concerns.
It also explores deep copies in polymorphic hierarchies and proper resource management.
All exercises follow the **C++98 standard**.

## 🧠 Key Concepts

- Virtual functions
- Runtime polymorphism
- Abstract classes
- Interfaces (pure abstract classes)
- Virtual destructors
- Deep copy in inheritance hierarchies
